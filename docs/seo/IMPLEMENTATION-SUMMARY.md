## Common Questions

### Q: How long does it take for pages to be indexed?
**A:** Typically 24-48 hours for new pages after requesting indexing. For automatic discovery, it can take 1-4 weeks.

### Q: Do I need to submit every UMKM profile individually?
**A:** No. The sitemap includes all published profiles. Search engines will discover them automatically. However, you can request indexing for important profiles to speed up the process.

### Q: What if my sitemap shows errors?
**A:** Follow the troubleshooting guide in `search-engine-submission-guide.md`. Common issues include invalid XML format, inaccessible URLs, or server errors.

### Q: How often should I check indexing status?
**A:** Daily for the first 2 weeks, then weekly for ongoing monitoring. Use the quick reference guide for the weekly checklist.

### Q: Can I automate the monitoring?
**A:** Yes. See the "Automated Monitoring Setup" section in `monitoring-dashboard-template.md` for options including Google Sheets integration and email alerts.

### Q: What if pages aren't being indexed?
**A:** Check the troubleshooting section in the main guide. Common causes include robots.txt blocking, noindex tags, low-quality content, or being a new website.
