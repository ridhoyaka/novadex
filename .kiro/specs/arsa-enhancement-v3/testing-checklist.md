# ARSA v3.0 Dashboard Copy - Quick Testing Checklist

**Purpose:** Quick reference for testing dashboard copy with UMKM users  
**Use this during testing sessions**

---

## Pre-Session Setup ✓

- [ ] Test account prepared (30%, 60%, 90% completion levels)
- [ ] Screen recording software ready
- [ ] Note-taking materials ready
- [ ] Consent form signed
- [ ] Participant comfortable and ready
- [ ] Recording started

---

## Testing Phases

### 1. First Impressions (5 min)
- [ ] Show dashboard without explanation
- [ ] Ask: "Apa kesan pertama Anda?"
- [ ] Ask: "Apa yang paling menarik perhatian?"
- [ ] Ask: "Apakah ada yang membingungkan?"
- [ ] Note spontaneous reactions

### 2. Comprehension Check (15 min)

#### Profile Completion Section
- [ ] Ask: "Apa arti 'Kelengkapan Profil'?"
- [ ] Ask: "Apa arti persentase ini?"
- [ ] Ask: "Apa perbedaan Profil Dasar/Lengkap/Optimal?"
- [ ] Verify: No confusion about technical terms

#### Missing Fields List
- [ ] Ask: "Apa yang ditunjukkan daftar ini?"
- [ ] Ask: "Apakah setiap item jelas?"
- [ ] Check: Understands "opsional" = optional
- [ ] Check: Understands "50 karakter" = character count

#### Status Indicators
- [ ] Ask: "Apa arti 'Profil Aktif' vs 'Tidak Aktif'?"
- [ ] Ask: "Bagaimana cara mengubah status?"
- [ ] Check: Understands publish/unpublish

#### Help Text
- [ ] Point to help icons (ℹ️)
- [ ] Ask: "Apakah teks bantuan membantu?"
- [ ] Check: Language is clear and friendly

#### Quick Actions
- [ ] Ask: "Apa fungsi tiga kotak ini?"
- [ ] Check: Understands each action
- [ ] Check: Labels are clear

### 3. Task Completion (10 min)

**Task 1: Find Missing Info**
- [ ] Give task: "Profil Anda 50% lengkap. Apa yang kurang?"
- [ ] Success: Finds completion section
- [ ] Success: Reads missing fields
- [ ] Success: Knows what to do next

**Task 2: Edit Profile**
- [ ] Give task: "Tambahkan foto usaha. Bagaimana caranya?"
- [ ] Success: Clicks "Lengkapi" or "Perbarui Profil"
- [ ] Success: No hesitation

**Task 3: View Public Profile**
- [ ] Give task: "Lihat profil seperti yang dilihat pelanggan"
- [ ] Success: Finds "Halaman Publik"
- [ ] Success: Clicks "Buka Halaman"

**Task 4: Understand Flags** (if applicable)
- [ ] Give task: "Admin memberi catatan. Di mana?"
- [ ] Success: Finds flag notification
- [ ] Success: Understands message

### 4. Language Check (5 min)
- [ ] Ask: "Ada kata yang tidak dipahami?"
- [ ] Ask: "Bahasa terasa ramah atau terlalu formal?"
- [ ] Ask: "Ada 'bahasa komputer' yang terlalu teknis?"
- [ ] Ask: "Apakah pesan memotivasi Anda?"
- [ ] Verify: NO technical jargon (SEO, metadata, slug)

### 5. Feedback (5 min)
- [ ] Ask: "Apa yang paling Anda sukai?"
- [ ] Ask: "Apa yang paling membingungkan?"
- [ ] Ask: "Ada informasi yang kurang?"
- [ ] Ask: "Ada informasi yang tidak perlu?"
- [ ] Ask: "Jika bisa ubah satu hal, apa itu?"

## Critical Issues to Watch For

### ❌ RED FLAGS (Must Fix)
- [ ] User doesn't understand main purpose
- [ ] Technical jargon causes confusion
- [ ] Cannot complete basic tasks
- [ ] Privacy concerns raised
- [ ] Feels intimidated or overwhelmed

### ⚠️ YELLOW FLAGS (Should Fix)
- [ ] Hesitation on specific terms
- [ ] Multiple attempts to find information
- [ ] Asks for clarification repeatedly
- [ ] Expresses frustration
- [ ] Suggests simpler alternatives

### ✅ GREEN FLAGS (Good Signs)
- [ ] Immediate understanding
- [ ] Positive emotional response
- [ ] Completes tasks confidently
- [ ] Feels motivated to complete profile
- [ ] Would recommend to others

## Key Questions to Ask

**Comprehension:**
1. "Apa yang dimaksud dengan [term]?"
2. "Mengapa [feature] penting menurut Anda?"
3. "Apa yang akan Anda lakukan setelah melihat ini?"

**Usability:**
1. "Bagaimana cara [action]?"
2. "Di mana Anda mencari [information]?"
3. "Apa yang Anda harapkan terjadi jika [action]?"

**Language:**
1. "Apakah ada kata yang tidak Anda pahami?"
2. "Apakah bahasa terasa ramah atau formal?"
3. "Apakah ada istilah yang terlalu teknis?"

**Motivation:**
1. "Apakah pesan ini memotivasi Anda?"
2. "Apakah Anda merasa dihargai?"
3. "Apakah Anda ingin melengkapi profil setelah melihat ini?"

## Success Targets

**Minimum to Pass:**
- ✅ 80% task completion rate
- ✅ 90% comprehension ("Clear" ratings)
- ✅ 0% technical jargon confusion
- ✅ 80% overall satisfaction

**If targets not met:**
- Identify specific issues
- Prioritize by severity
- Plan fixes before next round
- Re-test with 2-3 participants

## Quick Reference: Dashboard Elements to Test

### Section 1: Profile Completion
- "Kelengkapan Profil Anda" heading
- Percentage display (e.g., "60%")
- Status badge ("Profil Dasar/Lengkap/Optimal")
- Progress bar with color coding
- Motivational message
- Help icon tooltip

### Section 2: Missing Fields
- "Tingkatkan profil Anda dengan melengkapi:" heading
- Bulleted list of missing items
- "Lengkapi Sekarang" button
- Success message (if complete)

### Section 3: Flags (if present)
- "Perhatian dari Admin" heading
- Flag type labels
- Reason text
- "Perbarui Profil" link

### Section 4: Quick Actions
- "Perbarui Profil" card
- "Profil Aktif/Tidak Aktif" card
- "Halaman Publik" card
- Icons and descriptions

### Section 5: Profile Preview
- "Tampilan Profil Anda" heading
- Logo/placeholder
- Business name, category, district
- Description preview
- Help icon tooltip

---

## Terms That MUST Be Clear

✅ **Should be understood:**
- Kelengkapan Profil
- Profil Dasar/Lengkap/Optimal
- Profil Aktif/Tidak Aktif
- Opsional
- Minimal X karakter
- Halaman Publik

❌ **Should NOT appear:**
- SEO
- Metadata
- Slug
- Optimization
- Schema
- Indexing

---

**Quick Tip:** If participant asks "What does [term] mean?", that's a sign the copy needs improvement!

**Remember:** We're testing the system, not the user. Every confusion is valuable feedback.
